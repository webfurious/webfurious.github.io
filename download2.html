StudioFlow Alpha Release — Installation Instructions

Features Available for Testing
This is an early build, so only a core subset of features is currently available and functional.
The app focuses on simple, effective business management for freelancers and small creative businesses.

Currently Available Sections (Fully Functional with CRUD Operations)

Client Management
— Create new clients
— View client details
— Edit existing client information
— Delete clients

Project Management
— Create new projects (with client auto-complete/lookup from the Client database)
— View project details
— Edit project information
— Delete projects

Invoice Management
— Create new invoices (with client and project auto-complete/lookup)
— View invoice details
— Edit invoice information
— Delete invoices

Dashboard Overview
Displays recent activity (last 5 items) for clients, projects, and invoices
Shows due and overdue invoices with basic highlighting

Important Notes for Alpha Testers
This is a very early alpha — many sections (e.g., full scheduling/tasks, financial summaries, reporting, advanced settings) are still under development and not yet available.
All data is stored securely in a private Firebase backend, but please avoid entering sensitive real-world information during testing.
Bugs, crashes, and incomplete features are expected. Your feedback is incredibly valuable.

Download the APK

Click the link below to download the installation file:
StudioFlow — Alpha 0.01 (APK)

Install the APK

Locate the downloaded file (usually in your Downloads folder) and tap it to begin installation.

If prompted, enable Install unknown apps or Allow from this source for your browser or file manager.

Typical path:
Settings → Apps → Special app access → Install unknown apps → select browser/file manager → Allow

Security Note

Android may warn that the app is from an untrusted source. This is normal for non-Play Store alpha builds. The APK is built by me, but feel free to scan it for peace of mind.
