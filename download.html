StudioFlow Alpha Release — Installation Instructions

Features Available for Testing
This is an early build, so only a core subset of features is currently available and functional.
The app focuses on simple, effective business management for freelancers and small creative businesses.

Currently Available Sections (Fully Functional with CRUD Operations)

Client Management
—Create new clients
—View client details
—Edit existing client information
—Delete clients

Project Management
—Create new projects (with client auto-complete/lookup from the Client database)
—View project details
—Edit project information
—Delete projects

Invoice Management
—Create new invoices (with client and project auto-complete/lookup)
—View invoice details
—Edit invoice information
—Delete invoices

Dashboard Overview
Displays recent activity (last 5 items) for clients, projects, and invoices
Shows due and overdue invoices with basic highlighting

Important Notes for Alpha Testers
This is a very early alpha — many sections (e.g., full scheduling/tasks, financial summaries, reporting, advanced settings) are still under development and not yet available.
All data is stored securely in a private Firebase backend, but please avoid entering sensitive real-world information (e.g., full payment details, personal IDs) during testing.
Bugs, crashes, and incomplete features are expected. If you encounter any issues, please report them with screenshots, steps to reproduce, and device details (Android version, model, etc.). Your feedback is incredibly valuable! Thank you again for your help in shaping StudioFlow.

This is an early alpha version, so please note the following steps and important information:

Download the APK

Click the link to download the installation file: StudioFlow—Aplha 0.01

Install the APK

Locate the downloaded file (usually in your Downloads folder) and tap it to begin installation.

If prompted, enable "Install unknown apps" or "Allow from this source" in your browser or file manager (this is required because the APK is not yet signed by Google Play).

On most Android devices:
Settings → Apps → Special app access → Install unknown apps → select your browser or file manager → toggle "Allow from this source" on.

Security Note
Since this is an unsigned alpha build, Android may display a warning that the app is from an untrusted source. This is normal for non-Play Store APKs. The file is safe (it's built by me), but always scan with antivirus if you prefer extra peace of mind.

Troubleshooting
If you encounter any installation issues (e.g., error messages, blocked install, or device-specific restrictions), please let me know with details (Android version, device model, exact error message, screenshot if possible).
